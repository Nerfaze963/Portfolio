Présentation


realisé en cours de l'année 2023.

L'application de réservation de trajets Sicilylines a été réalisé avec l'utilisation d'une machine virtuel.
Le projet a été réalisé avec le framework Symfony

Afin de de créer le site, nous avons dû creer une base de données avec les différentes entité qui la compose.


Le projet est la mise en place d'un site de réservation. Une personne peut se connecter ou s'inscrire afin de réserver une traversée. Des formulaires ont été creer afin de permettre d'ajouter, de modifier et de supprimer diverses informations directement dans la base de données
