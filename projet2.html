Application logicielle Sicilylines

Présentation


realisé en cours de l'année 2022.

L'application logicielle Sicilylines a été réalisé avec l'utilisation de l'éditeur de code Visual studio 2022.
Le projet a été réalisé en language C# et des requête SQL car nous avons eu besoin de creer une base de données sur MySQL à l'aide des ressources fournit par les professeurs.

L'application Sicilylines à été réalisé en groupe. Le code permet d'ajouter, de supprimer et de modifier une liaison d'un port a un autres. Lorqu'une des 3 fonctions est utilisé, la base de données le prend en compte et ajoute, supprime ou modifie une liaison automatiquement.

Les fonctions de l'application Sicilylines


L'ajout

la liaison du port de départ et du port d'arrivé est directement ajouté à la base de données


La suppression

La suppression s'effectue directement à la ligne sélectionné dans la base de données et supprimer les liaisons existantes


La modification

La modification permet de modifier l'heure d'une liaison

La table liaison indique plusieurs champs tel que :

- La durée de la traversée
- Le port de départ
- Le port d'arrivé
- L'ID du secteur affecté

Le nom des port d'arrive et des seteurs est récuperer dans leur tables respectives
